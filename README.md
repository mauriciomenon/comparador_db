# ComparadorExcel

Algoritmo que compara duas tabelas no excel com base nas colunas 3 e 4 e cria um arquivo .xlsx contendo: <br />
Linhas excluídas da primeira tabela;<br />
Linhas adicionadas à segunda tabela;<br />
Linhas que possuem as colunas 3 e 4 iguais porem com outros campos diferentes.<br />
